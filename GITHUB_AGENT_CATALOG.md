## 🔑 Key Takeaways für PocketCodeAgent

1. **SKILL.md ist der neue Standard** — Microsoft + VoltAgent etablieren einheitliche Skill-Definitionen
2. **MCP (Model Context Protocol) dominiert** — alle modernen Tools sprechen MCP
3. **Coordinator/Worker ist die bewährte Architektur** — Research → Synthesis → Implementation → Verification
4. **Prompt Injection Defense wird kritisch** — NVIDIA Garak + ChatGPT Agent Mode Patterns
5. **Agent Memory ist Pflicht** — Cross-Session-Persistenz via Room DB ist korrekt
6. **Subagent-Spezialisierung > Generalisten** — 11 Rollen decken die wichtigsten Domains ab
7. **Verification-First** — nie Erfolg annehmen ohne faktische Prüfung
